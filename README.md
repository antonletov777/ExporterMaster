# Что я реализовал (по шагам)
# 1. Разработка микросервиса на Spring Boot

- Создал проект с нуля.

- Настроил окружение и подключил зависимости
  
- Реализовал REST-endpoint для скачивания CSV-файла.


# 2. Интеграция с REST API киберспортивного портала

- Изучил Swagger-документацию API.

Реализовал HTTP-клиент, который:

- отправляет запросы к внешнему API,

- получает и десериализует JSON-ответы,

- преобразует данные в DTO,

- корректно обрабатывает ошибки сети и недоступность API.

# 3. Создание моделей и DTO

Анализировал структуру входных данных и создал данные классы:

- PlayerDto

- TeamDto

- CountryDto

- MatchHistoryDto

- ReportRow

- ResponseWrapper

Настроил преобразование данных между слоями через сервисы.


# 4. Генерация CSV-отчёта

Написал собственный CSV-генератор, который:

- формирует заголовки столбцов,

- экранирует спецсимволы,

- добавляет вычисляемое поле fullName,

- располагает колонки в нужном порядке,

- отдаёт CSV-файл пользователю.


# Результат

Я разработал полноценный микросервис, который:

- стабильно получает данные из стороннего API,

- преобразует и нормализует их,

- вычисляет итоговое поле fullName,

- формирует корректный CSV-отчёт,

-  предоставляет результат через REST-endpoint.
